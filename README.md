# Job Portal Frontend 🚀

A modern, responsive Job Portal frontend built using **React.js** and **Tailwind CSS**, featuring job listings, search & filters, dashboard UI, routing, and pixel-perfect design converted from Figma.

## 🔥 Features
- Job listings with search and filters
- Job details page with apply modal
- User dashboard with statistics
- Responsive UI (mobile & desktop)
- Figma to UI conversion
- React Router based navigation

## 🛠 Tech Stack
- React.js (Vite)
- Tailwind CSS
- React Router
- Lucide Icons
- Vercel (Deployment)
